# chapter-core-package

## Purpose

**REMOVED** — The `@clawmasons/chapter-core` package has been eliminated. Its components (apps, tasks, skills) are now inlined directly into the `templates/note-taker/` directory with `{{projectScope}}` placeholders, making initialized projects fully self-contained. See `inline-template-components` spec for the replacement.

## Requirements

*All requirements removed. See change `remove-chapter-core` for migration details.*
