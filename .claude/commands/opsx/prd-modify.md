# .claude/commands/opsx/prd-modify.md
---

1. Switch to Plan mode
2. **Find PRD to modify** Find the relevant PRD.md in the openspce/prds/ directory.  
3. **Find IMPLEMENTATION plan** Find if there is already a IMPLEMENTATION.md with the prd.  If so, the unapplied changes will need to be modified.  If a relevant change has already been applied (has links to the archived change), then you will need to add a new change to the IMPLEMENTATION
4. Review project structure for more relevant information, including all specs found in openspec/**/*.spec


Create a summary of the changes you plan to make to the PRD.md and IMPLEMENTATION.md

Once you confirms.  continue with the change.
